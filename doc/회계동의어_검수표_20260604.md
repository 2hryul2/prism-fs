# 회계 동의어 사전 — 회계팀 검수표

> 작성: 2026-06-04 · 대상: `src/synonyms.py` 동의어 그룹 · 검수자: 신한금융지주 회계팀
> 목적: 주석 검색(BM25/lexical 확장)의 **재현율↑**. 동의어는 **임베딩 질의엔 미적용**(원질의 유지),
> BM25/lexical 텍스트에만 덧붙음 → 정밀도 훼손 최소. **숫자 무경유·결정론**.
> 근거: 2026Q1 4사(신한·KB·하나·우리) 주석 제목 실측 + K-IFRS 일반 용어.

## 검수 방법
- **반영여부=반영**: 이미 `synonyms.py`에 적용됨(코퍼스 실측·동일 개념 확실).
- **반영여부=검수대기**: 위험/판단 필요로 **미적용**. 아래 `검수결과`에 **승인/보류/수정**을 기입해 주세요.
- **오확장 위험**: 의미가 다른 용어를 잘못 묶으면 오답 유발 → 해당 컬럼 확인.
