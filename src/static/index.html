🤖 주석 자연어 질의 (AI 보조 · 인용 강제)

4사 주석 텍스트에 자연어로 질의합니다. 숫자는 생성하지 않으며(정량은 재무제표 비교 탭), 답변은 항상 출처(회사·주석번호·페이지)를 동반합니다. 로컬 Ollama 옵트인 — 미가동 시 검색 결과만 표시.

ℹ️ 표·숫자는 원본 PDF 그래픽을 그대로 렌더링하며 AI는 숫자를 재구성·환산하지 않습니다. AI 답변(보라 박스)은 주석 텍스트 기반이며 출처 인용을 동반합니다(human-in-the-loop 검증 필요).

📈 재무제표 비교 결정론

연결(CFS)/별도(OFS) 재무제표를 원문 그대로 비교합니다. 증감·차이·비율 등 파생값은 결정론 계산이며 각 값에 ⓘ 계산식 근거가 붙습니다. (단위 환산·AI 재구성 없음)

🔒 원문 보존(thstrm/frmtrm/frmtrm_q) · 단위 환산 0 · 파생값은 결정론 계산 + 계산식 provenance · 출처 DART fnlttSinglAcntAll → fs_structured.json.

🧭 커버리지·공시갭 신한 중심

4사 기준. 주석 "구조"만 비교하며 숫자·단위는 재구성/환산하지 않습니다.

A. 횡단 커버리지 매트릭스

주제 × 회사 → 전용 주석 / 통합·관련 / 미발견. 전용·관련 셀을 더블클릭하면 비교 탭에서 4사 전체를 해당 주제로 검색합니다.

전용 주석 통합·관련 미발견

B. 신한 vs 동종 주석 차집합

base=신한, peers=KB·하나·우리. 의미(임베딩) 매칭 임계값 0.62 기준. 제목이 달라도 의미가 같으면 대응으로 간주합니다.

C. 주제별 4사 정렬 (표준 주제 매핑)

회사마다 번호·제목이 다른 주석을 canonical 주제로 분류해 4사를 나란히 정렬(임베딩, AI 무경유). 칩 클릭 시 원본 PDF.

"주제 매핑 실행"을 눌러 4사를 표준 주제로 정렬하세요.

D. AI 비교 메모 초안 옵트인·인용 강제

선택 주제에 대해 4사 주석의 회계정책·가정 차이를 AI가 초안으로 정리(로컬 Ollama 옵트인). 숫자 생성 없음, 문장별 출처 동반, 검토 필요.

ℹ️ 커버리지·차집합·주제 매핑 분류는 임베딩 코사인 + 규칙으로만 산출되며 AI가 주석/숫자를 생성하지 않습니다. AI 비교 초안(D)은 주석 텍스트 기반·출처 인용 동반이며 담당자 검토가 필요합니다. (연결/별도 동일 축끼리 비교)
